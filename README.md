##  📣 개요
> **회사 내 활발한 네트워킹을 촉진시키는 실시간 랜덤 매칭 서비스입니다.**  
  > 💬 실시간 매칭 후 채팅을 통해 온/오프라인 만남 제공  
  > 💬 같은 회사 내 타 부서와의 교류를 통한 친목, 인사이트 제공 및 스트레스 해소
